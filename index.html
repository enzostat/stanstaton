Krang!

Krang! is an online music sequencer with a theremin addition. If you are looking to play around with musical elements and get used to a music sequencer, this is the project for you. This won the General Assemblies Geek Out hackathon.

Stack: React App using Tone.js


Seattle AirB&B

Seattle AirB&B is an attempt to replicate the functionality of AirB&B in a MERN stack. This site allows you to sign up and search through available properties by neighborhoods in Seattle. Once you've found a property you like, you can book it and see your booked properties in your profile.

Stack: MongoDB, Express, Node.js, React using Moment.js, Axios and JSON Web Token


My Budget App

My Budget App allows a user to track their expenses. Once a user has entered their salary and monthly expenses, they can also add incidental expenses.

Stack: MongoDB, Express, Node.js, React with Google Charts for React
